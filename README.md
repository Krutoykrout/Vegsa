# VEGAS — сайт каталога автомобилей под выкуп

Статический сайт для GitHub Pages.

## Структура

- `index.html` — главная страница.
- `catalog.html` — каталог с фильтрами, сортировкой и кнопкой «Показать ещё».
- `cars/*.html` — отдельные HTML-страницы автомобилей с готовым основным контентом для SEO.
- `assets/data/cars.js` — данные автомобилей для каталога и фильтров.
- `robots.txt` и `sitemap.xml` — базовая SEO-разметка для индексации.

Сайт без административной панели. Изменения вносятся в файлы проекта.
